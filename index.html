Peking University · CS PhD Candidate

Hongcheng Wang

I am a fourth-year CS PhD candidate at Peking University. My supervisor is Prof. Hao Dong.

My current research interests are organized into two parts:

  • Chain-of-thought Generation: I study reinforcement-learning post-training for large models, especially how to stably train <think> to better guide <answer>.
  • Human-centered Robot Decision Making: I take a Cognitive Behavioral Theory's perspective to enable robots to understand human cognition (e.g., latent demands and preferences) and behavior (e.g., observable habits and norms), so they can better serve people.

Research

My current work has two main directions: (1) chain-of-thought reasoning for large models, and (2) cognitively grounded, human-centered robot decision making in embodied settings, where robots model human demands, preferences, habits, and norms to provide better assistance.

User-Centric Object Navigation: A Benchmark with Integrated User Habits for Personalized Embodied Object Search

Hongcheng Wang*, Jinyu Zhu*, Hao Dong

ICRA, 2026

Paper · Code

We introduce User-centric Object Navigation (UcON), a benchmark with integrated user habits for personalized embodied object search and habit-conditioned planning.

Why Tree-Style Branching Matters for Thought Advantage Estimation in GRPO

Hongcheng Wang*, Yinuo Huang*, Sukai Wang, Guanghui Ren, Hao Dong

arXiv preprint, 2025

Paper

We analyze variance in thought-level advantage estimation for GRPO and show why answer-level branching is key for stable and efficient reasoning optimization.

MO-DDN: A Coarse-to-Fine Attribute-based Exploration Agent for Multi-Object Demand-driven Navigation

Hongcheng Wang*, Peiqi Liu*, Wenzhe Cai, Mingdong Wu, Zhengyu Qian, Hao Dong

NeurIPS, 2024

Project · Paper · Code · Video

The extended version of DDN. We propose a multi-object demand-driven navigation benchmark and train a coarse-to-fine attribute-based exploration agent to solve this task.

Find What You Want: Learning Demand-conditioned Object Attribute Space for Demand-driven Navigation

Hongcheng Wang, Andy Guan Hong Chen, Xiaoqi Li, Mingdong Wu, Hao Dong

NeurIPS, 2023

Project · Paper · Code · Video

The first Demand-driven Navigation paper. We propose a task that requires an agent to find an object satisfying a user demand in an environment.

Learning Semantic-Agnostic and Spatial-Aware Representation for Generalizable Visual-Audio Navigation

Hongcheng Wang*, Yuxuan Wang*, Fangwei Zhong, Mingdong Wu, Jianwei Zhang, Yizhou Wang, Hao Dong

RA-L, 2023

Project · Paper · Code · Video

We propose a semantic-agnostic and spatial-aware representation for generalizable visual-audio navigation, improving baseline performance.
