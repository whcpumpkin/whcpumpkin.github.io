Hongcheng Wang

I am a fourth-year CS PhD candidate at Peking University. My supervisor is Prof. Hao Dong. My current research interests are organized into two parts:

(1) chain-of-thought reasoning: I study reinforcement-learning post-training for large models, especially how to stably train <think> to better guide <answer>.

(2) Human-centered robot decision making: I focus on aligning human intentions, motivations, and demands with robust embodied decision making and robot actions.

Research

User-Centric Object Navigation: A Benchmark with Integrated User Habits for Personalized Embodied Object Search
Hongcheng Wang, Jinyu Zhu, Hao Dong
ICRA, 2026
Paper / Code

We introduce User-centric Object Navigation (UcON), a benchmark with integrated user habits for personalized embodied object search and habit-conditioned planning.

Why Tree-Style Branching Matters for Thought Advantage Estimation in GRPO
Hongcheng Wang, Yinuo Huang, Sukai Wang, Guanghui Ren, Hao Dong
arXiv preprint, 2025
Paper

We analyze variance in thought-level advantage estimation for GRPO and show why answer-level branching is key for stable and efficient reasoning optimization.

MO-DDN: A Coarse-to-Fine Attribute-based Exploration Agent for Multi-Object Demand-driven Navigation

NeurIPS, 2024
Project / Paper / Code / Video

The extended version of DDN.

Project / Paper / Code / Video

The First Demand-driven Navigation Paper.
